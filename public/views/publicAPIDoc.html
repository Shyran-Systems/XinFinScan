Introduction


We will show all the Public APIs here.

The APIs support GET/POST requests .

You can request by a rate limit of 3 requests/sec , otherwise you will be blocked.

Each API return data in format like :{"status":1, "message":"OK", "result":""}.
status: 1 mean Success , 0 mean Fail
message: OK or fail reason
result: json format data about the request result


Both http and https are supported

Balance APIs


Get total XDC on block chain 

  • http://localhost:3000/publicAPI?module=balance&action=totalXDC&apikey=YourApiKeyToken

Get XDC Balance for a single Address

  • http://localhost:3000/publicAPI?module=account&action=balance&address=0xeb0e5b3c177c72235d07d8daa6e7ad8fe701e24a&tag=latest&apikey=YourApiKeyToken

Get XDC Balance for multiple Addresses in a single call 
  • http://localhost:3000/publicAPI?module=account&action=balancemulti&address=0xddbd2b932c763ba5b1b7ae3b362eac3e8d40121a,0x63a9975ba31b0b9626b34300f7f627147df1f526,0x198ef1ec325a96cc354c7266a038be8b5c558f67&tag=latest&apikey=YourApiKeyToken
  • Separate addresses by comma, up to a maxium of 20 accounts in a single batch

Contract APIs


Newly verified Contracts are synced to the API servers within 5 minutes or less

Get Contract ABI for Verified Contract Source Codes 

  • http://localhost:3000/publicAPI?module=contract&action=getabi&address=0xc01fbdf458041359a8eeaf03d9991dd8fd8dbd6f&apikey=YourApiKeyToken

A simple sample for retrieving the contractABI using Web3.js and Jquery to interact with a contract

var Web3 = require('xdc3');;
    var web3 = new Web3(new Web3.providers.HttpProvider());
    var version = web3.version.api;
                
    $.getJSON('http://localhost:3000/publicAPI?module=contract&action=getabi&address=0xc01fbdf458041359a8eeaf03d9991dd8fd8dbd6f', function (data) {
        var contractABI = "";
        contractABI = JSON.parse(data.result);
        if (contractABI != ''){
            var MyContract = web3.eth.contract(contractABI);
            var myContractInstance = MyContract.at("0xc01fbdf458041359a8eeaf03d9991dd8fd8dbd6f");
            var result = myContractInstance.getBuyPrice();
            console.log("result : " + result); 
        } else {
            console.log("Error" );
        }            
    });

Event Logs


[Beta] The Event Log API was designed to provide an alternative to the native eth_getLogs. Below are the list of supported filter parameters:

  • param address: contract address
  • param[option] fromBlock: logs in block is great than or equle fromBlock
  • param[option] toBlock: logs in block is little than or equle toBlock
  • param[option] topics: topics filter. multi topic separate with comma.
    such as: ...&topics=value0,,value2&... this exprission will fetch logs that topic0 is value0 and topic2 is value2.
    each value's length is 66(contain:0x)
  • param[option] data: log data(not index param data)
  • param[option] returnFilters: return partial data(full data as default). multi topic separate with comma.
    such as: ...&returnFilters=address,timestamp,topics,data&...
fromBlock and toBlock accepts the blocknumber (integer, NOT hex) or 'latest' (earliest & pending is NOT supported yet)
  • http://localhost:3000/publicAPI?module=logs&action=getLogs&address=0x866171faf25577a8ba61645d31ffae70047f73f6&topics=0xddf252ad1be2c89b69c2b068fc378daa952ba7f163c4a11628f55a4df523b3ef,,0x000000000000000000000000259b357e09f55698030b30f5b9fc5990916b9a5c&returnFilters=timestamp,topics,-_id&apikey=YourApiKeyToken

Geth/Parity Proxy APIs


The following are the limited list of supported Proxied APIs for Geth available through XDCScan. For the list of the parameters and descriptions please see https://github.com/ethereum/wiki/wiki/JSON-RPC. Parameters provided should be named like in the examples below. For compatibility with Parity, please prefix all hex strings with "0x"

eth_blockNumber
Returns the number of most recent block

  • http://localhost:3000/publicAPI?module=proxy&action=eth_blockNumber&apikey=YourApiKeyToken


eth_getBlockByNumber
Returns information about a block by block number.

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getBlockByNumber&blockNumber=666&apikey=YourApiKeyToken


eth_getBlockTransactionCountByNumber
Returns the number of transactions in a block from a block matching the given block number

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getBlockTransactionCountByNumber&blockNumber=666&apikey=YourApiKeyToken


eth_getTransactionByHash
Returns the information about a transaction requested by transaction hash

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getTransactionByHash&txhash=0x1e2910a262b1008d0616a0beb24c1a491d78771baa54a33e66065e03b1f46bc1&apikey=YourApiKeyToken


eth_getTransactionByBlockNumberAndIndex
Returns information about a transaction by block number and transaction index position

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getTransactionByBlockNumberAndIndex&blockNumber=1267471&index=0x0&apikey=YourApiKeyToken


eth_getTransactionCount
Returns the number of transactions sent from an address

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getTransactionCount&address=0x2910543af39aba0cd09dbb2d50200b3e800a63d2&tag=latest&apikey=YourApiKeyToken


eth_sendRawTransaction
Creates new message call transaction or a contract creation for signed transactions

  • http://localhost:3000/publicAPI?module=proxy&action=eth_sendRawTransaction&hex=0xf904808000831cfde080&apikey=YourApiKeyToken
    • (Replace the hex value with your raw hex encoded transaction that you want to send.
    Send as a POST request, if your hex code is particularly long)


eth_getTransactionReceipt
Returns the receipt of a transaction by transaction hash

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getTransactionReceipt&txhash=0x1e2910a262b1008d0616a0beb24c1a491d78771baa54a33e66065e03b1f46bc1&apikey=YourApiKeyToken


eth_call
Executes a new message call immediately without creating a transaction on the block chain

  • http://localhost:3000/publicAPI?module=proxy&action=eth_call&to=0xAEEF46DB4855E25702F8237E8f403FddcaF931C0&data=0x70a08231000000000000000000000000e16359506c028e51f16be38986ec5746251e9724&blockNumber=latest&apikey=YourApiKeyToken


eth_getCode
Returns code at a given address

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getCode&address=0xf75e354c5edc8efed9b59ee9f67a80845ade7d0c&vs=latest&apikey=YourApiKeyToken


eth_getStorageAt (**experimental)
Returns the value from a storage position at a given address.

  • http://localhost:3000/publicAPI?module=proxy&action=eth_getStorageAt&address=0x6e03d9cce9d60f3e9f2597e13cd4c54c55330cfd&position=0x0&blockNumber=latest&apikey=YourApiKeyToken


eth_gasPrice
Returns the current price per gas in wei.

  • http://localhost:3000/publicAPI?module=proxy&action=eth_gasPrice&apikey=YourApiKeyToken


eth_estimateGas
Makes a call or transaction, which won't be added to the blockchain and returns the used gas, which can be used for estimating the used gas

  • http://localhost:3000/publicAPI?module=proxy&action=eth_estimateGas&to=0xf0160428a8552ac9bb7e050d90eeade4ddd52843&value=0xff22&gasPrice=0x051da038cc&gas=0xffffff&apikey=YourApiKeyToken
